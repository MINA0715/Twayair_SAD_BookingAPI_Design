    Method: CalendarFare (deprecated)

    Overview

    This gives the lowest fare in a day for each fare type.

    Service URL

    
https://<ServerURL:port>/twayair/service/postbook/CalendarFare

    Predecessor Method

    Nil

    Successor Method

    AirAvailability

    Description

    This operation retrieves the lowest fare in day for a given calendar period.

    Input

    The input of this method is the argument CalendarRetrieveFaresRQ having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    AgencyCodestringYes1Unique Identifier for the Travel. Max Length: 9
    OriginAirportstringYes1..1Onward origin airport code. Max Length: 5
    DestinationAirportstringYes1..1Onward destination airport. Max Length: 5
    StartDatestringYes1..1Onward date. YYYY-MM-DD
    EndDatestringYes1..1Return date. YYYY-MM-DD
    TripTypeTripTypeYes1..1OW-OneWay RT-RoundTrip MC-MultiCity Max Length: 2
    PointOfPurchasestringYes1..1Holds the point of purchase. Max Length: 3 (KR/JP/CN/TH/LA//GU/VN /MP/PH/KH/TW/MO//HK/MY)

    Output

    The output of this method is the argument CalendarRetrieveFaresRS having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    IsOnwardTimeViolatedbooleanYes1..1This is set to true if the onward time restriction is changed to ANYTIME.
    IsReturnTimeViolatedbooleanYes1..1This is set to true if the return time restriction is changed to ANYTIME
    OnwardBestFareDetailsBestFareDetailsTypeNo0..*Onward Fares. Populated if onward fares are present
    ReturnBestFareDetailsBestFareDetailsTypeNo0..*Return Fares. Populated if return fares are present
    ErrorTypeErrorTypeNo0..1The reason for error in case there is no data retrieval.
    IsFareRefreshNeededbooleanYes1..1This is set to true if the return fares need to be refreshed

    Exceptions

    Error CodeMessage
    AVAILABILITY_069Flight not found.
    AVAILABILITY_072Mileage not defined.

    FARES_209No accessible Fare Levels found

    FARES_662Return date is null

    FARES_664Onward end time is zero
    FARES_665Return end time is zero
    FARES_666Pax count is zero
    FARES_667Invalid onward origin airport
    FARES_668Invalid onward destination airport
    FARES_669Invalid return origin airport
    FARES_670Invalid return destination airport
    FARES_672No accessible farelevels found
    FARES_677Day range specified is exceeds the maximum limit
