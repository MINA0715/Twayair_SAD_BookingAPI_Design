    • Appendix 1 – Error Codes

    The Error Codes specified along with the services are the list of exceptions that will be thrown from these services. If any undocumented error code is returned then the t’wayair API will respond with a generic error code ‘WS_000’. The message of this generic error code will contain the detailed message of the Error that has occurred. The Runtime exceptions like Unexpected Server Error, Parse Exception etc are not documented and will not be mapped to the generic Error code as mentioned above.

  • Sample code
    • String strTest = "encryptTest"; String strEncrypt = ""; String strDecrypt = ""; strEncrypt = TwayAir.Common.BookingAPI.Seed.SeedUtil.Encrypt(strTest); strDecrypt = TwayAir.Common.BookingAPI.Seed.SeedUtil.Decrypt(strEncrypt);

    Use TwaySeed.jar (java 1.5 version)

  • Sample code
