    Method: CheckinAvailability

    Overview

    This operation will check whether check-in is possible for the flight.

    Service URL

    
https://<ServerURL:port>/twayair/service/checkin/CheckinAvailability

    Predecessor Method

    None

    Successor Method

    SearchGuest

    Description

    Input

    The input of this method is the argument CheckinAvailabilityRQ having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1Unique Identifier for a reservation. Max Length: 6
    AgencyCodeStringYes1Identifier for an agency if the agency category is Travel Agency. Max Length: 9

    Method: SearchGuest

    Overview

    This operation will check whether check-in is possible for the guest.

    Service URL

    
https://<ServerURL:port>/twayair/service/checkin/SearchGuest

    Predecessor Method

    CheckinAvailabliliy

    Successor Method

    CheckinSeatAvailability

    Description

    In accordance with policy, passengers are classified as cases where Check-in or seat assign only is

    possible.

    In case of seat assignment only, an exchange coupon will be issued and the exchange coupon URL will be

    displayed on the Boarding Pass URL field.

    Please bring the exchange coupon to the counter and exchange it for a boarding pass.

    Online check-in information : [https://www.twayair.com/app/serviceInfo/contents/1121](https://www.twayair.com/app/serviceInfo/contents/1121)

    Input

    The input of this method is the argument SearchGuestRQ having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1Unique Identifier for a reservation. Max Length: 6
    AgencyCodeStringYes1Identifier for an agency if the agency category is Travel Agency. Max Length: 9
    SegmentIdStringYes1The Segment Id to uniquely identify the segment. Max Length: 25
    LanguageCodeStringYes1Boarding pass language code. Max Length: 5 ko-KR : korean ja-JP : Japanese en-US : English th -TH: Thai vi-VN : Vietnamese zh-CN: Chinese zh-TW : SimplifiedChinese ru-RU : Russian
    GuestIdStringYes1..*Unique Identifier for a passenger. Max Length: 25

    Output

    The output of this method is the argument SearchGuestRS having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1..1Unique Number for the reservation record.
    FlightSegmentDetailsForCheckinFlightSegmentDetailsForCheckinNo0..1*check-in information in the segment.
    ErrorTypeErrorTypeNo0..1The reason for error in case there is no data retrieval.

    Exceptions

    Error CodeMessage
    CON008Security Error
    DCS_039Passenger(s) not found.
    RQ_ERR_DCS_001Number format incorrect.Integer needed for NumberOfDays.
    RQ_ERR_DCS_002Number format incorrect.Integer needed for NumberinParty.
    RQ_ERR_DCS_005Invalid number in party
    WS_001Mandatory data missing AirlineCode
    WS_034Mandatory data missing ChannelKey,ChannelType
    WS_035Mandatory data missing ChannelKey,ChannelCode
    WS_036Mandatory data missing ChannelKey
    WS_037Mandatory data missing TravelDate
    WS_1111The system is currently busy. Please try again.
    WS_751User not defined in the system.
    WS_790Mandatory data missing Tour Operator Number
    WS_791Mandatory data missing Record Locator
    WS_999PNR # is mandatory for searching other passengers in the PNR.

    Method: CheckinSeatAvailability

    Overview

    This service returns the seat map(layout) and seat availability for Check-in.

    Service URL

    
https://<ServerURL:port>/twayair/service/checkin/CheckinSeatAvailability

    Predecessor Method

    SearchGuest

    Successor Method

    CheckinGuest

    Description

    This service returns the seat map(layout) and seat availability for Check-in.

    LocationAttribute

    W – WINDOW

    E – EMERGENCY EXIT

    D – DOOR

    A – AISLE

    K – BULKHEAD

    L – LAVATORY

    O – OVERWING

    M – NO WINDOW

    I – LIMITED RECLINE

    G – LIMITED OXYGEN MASK

    FacilityAttribute

    S – SMOKING

    P – PREMIUM SEAT

    R – NON RECLINING

    U – UNCOMFORTABLE

    B – HANDICAPPED

    F – SEMI PREMIUM SEAT

    H – FRONT NORMAL SEAT

    C – NORMAL SEAT

    J – PET SEAT

    V – BUSINESS SEAT

    Q – BASSINET

    N – NON SMOKING

    Input

    The input of this method is the argument CheckinSeatAvailabilityRQ having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1Unique Identifier for a reservation. Max Length: 6
    AgencyCodeStringYes1Identifier for an agency if the agency category is Travel Agency. Max Length: 9
    SegmentIdStringYes1The Segment Id to uniquely identify the segment. Max Length: 25
    GuestIdStringYes1..*Unique Identifier for a passenger. Max Length: 25

    Output

    The output of this method is the argument CheckinSeatAvailabilityRS having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1..1Unique Number for the reservation record.
    SeatMapInformationSeatMapInformationNo0..1The information to display the seat map is returned.
    FlightSegmentDetailsForCheckinFlightSegmentDetailsForCheckinNo0..1*check-in information in the segment.
    ErrorTypeErrorTypeNo0..1The reason for error in case there is no data retrieval.

    Exceptions

    Error CodeMessage
    DCS_1351Seat Map Not Attached.
    PYM_FEE_25Mandatory Details are missing
    SAT_018This flight segment does not have the seat map created.
    SAT_044The ASR Date range has crossed.
    WS_001Mandatory data missing AirlineCode
    WS_101Mandatory Data Missing.AirlineCode.
    WS_1111The requested service is currently blocked by the Administrator.Please try again later or contact the Administrator
    WS_122Mandatory Data Missing.Booking Channel.
    WS_145Mandatory Data Missing.FlightIdentifier.
    WS_146Mandatory Data Missing.Segment.
    WS_149Mandatory Data Missing.ChannelType.
    WS_150Mandatory Data Missing.Channel.
    WS_151Mandatory Data Missing.Locale.
    WS_751User not defined in the system.
    DCS_1351Seat Map Not Attached.

    Method: CheckinGuest

    Overview

    This operation can perform a check-in for the guest.

    Service URL

    
https://<ServerURL:port>/twayair/service/checkin/CheckinGuest

    Predecessor Method

    SearchGuest

    CheckinSeatAvailability

    Description

    SearchGuest and CheckinSeatAvailability determine which passenger and wich seat are available.

    You can check-in by call CheckinGuest, entering the GuestId and Seat Number.

    The language Information input to the language Code to set a language to receive an email and SMS.

    If assignSeatAvail is TRUE, a seat number is required, and if you purchase a seat in advance, you do not have to the seat number.

    Input

    The input of this method is the argument SearchGuestRQ having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1Unique Identifier for a reservation. Max Length: 6
    AgencyCodeStringYes1Identifier for an agency if the agency category is Travel Agency. Max Length: 9
    SegmentIdStringYes1The Segment Id to uniquely identify the segment. Max Length: 25
    LanguageCodeStringYes1Boarding pass language code. Max Length: 5 ko-KR : korean ja-JP : Japanese en-US : English th -TH: Thai vi-VN : Vietnamese zh-CN: Chinese zh-TW : SimplifiedChinese ru-RU : Russian
    GuestInfoGuestInfoForCheckinYes1..*Passenger GuestId and Seat Number for Check-in

    Output

    The output of this method is the argument SearchGuestRS having the structure defined by the following table.

    NameTypeRequired?OccursDescription
    PnrNumberStringYes1..1Unique Number for the reservation record.
    FlightSegmentDetailsForCheckinFlightSegmentDetailsForCheckinNo0..1*check-in information in the segment.
    ErrorTypeErrorTypeNo0..1The reason for error in case there is no data retrieval.

    Method: UnCheckinGuest

    Overview

    This operation can perform a check-in for the guest.

    Service URL

    
https://<ServerURL:port>/twayair/service/checkin/UnCheckinGuest

    Predecessor Method

    CheckinGuest

    Description

    You can uncheckin by call UnCheckinGuest, entering the GuestId.

    The language Information input to the language Code to set a language to receive an email and SMS.
